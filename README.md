## 心情罐罐是什么

简单说，它就是一个用表情图案来记录心情的手帐类 App。

你不用每天写长篇大论，打开 App，点一个最符合当下状态的表情，加几句备注，配张图，一条心情记录就完成了。快的时候十秒钟。

内置了 40 多种表情图案，覆盖从"超级开心"到"真的很崩"各种状态，也支持长按替换成你自己的图片——如果你觉得自带的表情不够准确表达某种微妙感受，这个功能挺有用的。

除了手动输入，还可以用语音快速记录。讲完之后，App 会用 AI 自动识别你当时的情绪状态，并帮你润色成更通顺的文字表达。适合那种"脑子里有感受，但一时组织不出语言"的场景。

## 用着有什么感觉

记录本身是轻量的，但看数据的时候往往会有点触动。

心情罐罐提供心情日历、柱状图、环形图等多种可视化统计，你可以清楚地看出自己哪几天状态比较差、情绪波动的规律大概是什么。有时候你以为自己最近"还好"，但图表一出来才发现那段时间确实情绪洼地。

另外支持补录过去时间段的心情，不用担心有几天忘记记了——回想一下再补上就行，历史数据还是连贯的。

iCloud 多端同步也有，手机、iPad 之间数据自动同步，不会因为换设备就找不到过去的记录了。
